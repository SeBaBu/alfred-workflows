### Things Workflow

First keyword:

task:

and

\#tag1 #tag2 name [project name/area name] next monday/.../sunday::note

and

\#tag1 #tag2 name [project name/area name] on january/../december <day as number>::note

Second keyword

task2:

\<task> ::\<note>
